# 🌍 TripAdvisor Clone

A responsive front-end clone of the TripAdvisor homepage built using **HTML** and **CSS**.
This project replicates the layout and design of the original TripAdvisor website for learning and practice purposes.

---

## 🚀 Features

* 🧭 Navigation bar with logo and menu
* 🔍 Search section UI
* 🖼️ Hero section with image and overlay text
* 🏷️ Image labels (like usernames on images)
* 📱 Responsive design (basic)
* 🎨 Clean and modern UI inspired by TripAdvisor

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Positioning)

---

## 📂 Project Structure

```
tripadvisor-clone/
│── index.html
│── style.css
│── images/
```

---

## 💡 What I Learned

* How to structure layouts using Flexbox
* Positioning elements (absolute & relative)
* Creating UI components like navbar, cards, and overlays
* Improving styling and spacing
